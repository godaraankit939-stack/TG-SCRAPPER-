# Telegram Restricted Content Saver

A powerful Python tool to extract and save media from restricted Telegram channels.

### Features
- **Fast Extraction:** No downloading required; uses server-side copying.
- **Multiple Destinations:** Save to Bot, Saved Messages, or Custom Groups.
- **Session Support:** Auto-login feature for seamless usage.

### Installation
1. Clone the repo: `git clone https://github.com/godaraankit939-stack/TG-SCRAPPER-.git`
2. Install requirements: `pip install -r requirements.txt`
3. Run the tool: `python main.py`
